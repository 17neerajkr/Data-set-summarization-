# Historical Investment Analysis Dataset

This dataset contains historical annual returns of different investment types.
It includes data for Stocks, Treasury Bills (T.Bills), and Treasury Bonds (T.Bonds).

## Dataset Overview
- Covers multiple years starting from 1928
- Shows yearly return percentages
- Helps compare investment performance over time

## Columns Description
- Year: Represents the time period of investment
- Stocks: Annual return of stock market investments
- T.Bills: Short-term government securities returns
- T.Bonds: Long-term government bond returns

## Key Insights
- Stocks show high volatility with larger gains and losses
- T.Bills provide stable but lower returns
- T.Bonds offer moderate and relatively stable returns

## Purpose
- Analyze long-term investment trends
- Compare risk vs return across asset classes
- Useful for financial modeling and forecasting

## Use Cases
- Data analysis projects (Pandas, Power BI)
- Investment strategy evaluation
- Machine learning models for prediction

## Conclusion
This dataset is ideal for understanding financial market behavior and investment performance over time.
